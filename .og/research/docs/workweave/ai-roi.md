# AI ROI

> Measure the return on investment of your AI coding tools

The AI ROI report helps you understand the value your organization gets from AI coding tools.

## Who can see it?

Anyone with the **Team lead**, **Manager**, or **Admin** role can access the AI ROI report.

## What's included

### Cost tracking

* **Tool subscription costs** — Monthly spend on each AI tool
* **Cost per user** — Average cost per active user
* **Cost trends** — How spending changes over time

### Impact metrics

* **AI code percentage** — Portion of code written with AI assistance
* **Output correlation** — Relationship between AI usage and productivity
* **Efficiency gains** — Productivity improvements from AI adoption

### Usage patterns

* **Active users** — Team members using AI tools
* **Tool adoption** — Which tools are most popular
* **Usage trends** — How adoption changes over time

## Setting up cost tracking

To track AI tool costs, configure your subscriptions:

<Steps>
  <Step title="Go to settings">
    Navigate to **Organization settings** → **Financial**
  </Step>

  <Step title="Add subscriptions">
    Enter your subscription tier and seat count for each tool
  </Step>

  <Step title="View ROI">
    Return to the AI ROI report to see cost analysis
  </Step>
</Steps>
