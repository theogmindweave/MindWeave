# Slash commands

> Pre-built prompts for common tasks in Wooly AI

Slash commands are shortcuts in Wooly AI that run pre-built prompts for common tasks like performance reviews, 1:1 prep, and status updates.

## How to use

Type `/` in the Wooly AI chat to see available commands. Select a command and add any required mentions (like `@person` or `@team`).

## Built-in commands

| Command                | Description                                           |
| ---------------------- | ----------------------------------------------------- |
| `/performance-review`  | Generate a performance review for a team member       |
| `/one-on-one`          | Prepare talking points for a 1:1 meeting              |
| `/onboarding-analysis` | Analyze ramp-up for recent hires                      |
| `/weekly-update`       | Generate a weekly status update                       |
| `/monthly-update`      | Generate a monthly status update                      |
| `/quarterly-update`    | Generate a quarterly status update                    |
| `/coaching`            | Generate a 30-day coaching plan for an individual     |
| `/ai-impact`           | Analyze who is using AI most effectively              |
| `/quality-issues`      | Identify where quality is an issue and how to fix it  |
| `/highlight-successes` | Highlight recent successes and wins for an individual |
| `/highlight-failures`  | Identify areas of concern with specific examples      |

Most commands accept an optional `@person` or `@team` mention to scope the output.
