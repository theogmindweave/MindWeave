# Team

> Dive into a team's contributions and metrics

The Team report shows detailed metrics for a specific team.

## Who can see it?

* **Admin** and **Manager** roles can see any team's report
* **Team lead** role can see reports for teams they belong to (including child teams)
* **Member** role cannot access team reports

## Navigating to a team

There are several ways to view a team report:

1. Click a team name in the sidebar
2. Use the command menu (`⌘K`) and search for a team
3. Click on a team name in any report

## What's included

* **Team output** — Combined code output for all team members
* **Member breakdown** — Individual contributions within the team
* **PR activity** — Pull requests opened, merged, and reviewed
* **Code review metrics** — Review turnaround and depth
* **AI adoption** — AI tool usage across the team

## Team hierarchy

If your team has child teams, the report includes data from all descendant teams in the hierarchy.
