# GitHub

> Connect your GitHub repositories to Weave

Connect GitHub to sync your pull requests, commits, code reviews, and team structure.

## What's synced

* Pull requests and their metadata
* Commits and code changes
* Code reviews and comments
* Team members and organization structure
* Repository information

## Connecting GitHub

<Steps>
  <Step title="Start connection">
    Go to **Data** and click **Connect** on GitHub
  </Step>

  <Step title="Authorize Weave">
    Sign in to GitHub and authorize the Weave app
  </Step>

  <Step title="Select repositories">
    Choose which repositories to sync
  </Step>

  <Step title="Wait for sync">
    Initial sync may take several minutes depending on repository size
  </Step>
</Steps>

## Permissions required

Weave requests the following GitHub permissions:

* **Repository access** — Read access to code, commits, and pull requests
* **Organization access** — Read access to organization members and teams

## Multiple organizations

You can connect multiple GitHub organizations. Each organization's data will be synced separately.

## Sync frequency

Weave syncs GitHub data continuously. Changes typically appear within a few minutes.
