# Keyboard shortcuts

> Navigate Weave faster with keyboard shortcuts

Weave includes keyboard shortcuts to help you navigate and filter data quickly.

## Command menu

Press `âŒ˜K` (Mac) or `Ctrl+K` (Windows/Linux) to open the command menu. From here you can:

* Search for any report, team, or team member
* Apply filters quickly
* Toggle settings
* Ask Wooly AI a question

## Navigation shortcuts

Press `1` through `0` to jump to the corresponding report in your sidebar.

## Filter shortcuts

Quickly open filter menus from any dashboard page.

| Shortcut | Action                   |
| -------- | ------------------------ |
| `F`      | Open "Add filter" menu   |
| `G`      | Change time granularity  |
| `D`      | Change date filter       |
| `U`      | Change user filter       |
| `T`      | Change team filter       |
| `R`      | Change repository filter |
| `P`      | Change project filter    |
| `A`      | Change AI tool filter    |

## Action shortcuts

| Shortcut | Action                                          |
| -------- | ----------------------------------------------- |
| `S`      | Toggle left sidebar                             |
| `W`      | Toggle Wooly AI sidebar                         |
| `N`      | Start new chat (on agent page) or open Wooly AI |
| `C`      | Toggle customize mode                           |
| `Z`      | Switch between light and dark mode              |
