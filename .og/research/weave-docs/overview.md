# Overview

> A high-level view of your organization's engineering metrics

The Overview report provides a dashboard of key metrics across your entire organization.

## Who can see it?

Anyone with the **Team lead**, **Manager**, or **Admin** role can access the Overview report.

## What's included

The Overview report shows:

* **Code output** — Total engineering output across the organization
* **PR volume** — Pull requests opened and merged
* **Code review activity** — Reviews given and turnaround time
* **AI adoption** — AI code percentage and tool usage
* **Team comparisons** — Metrics broken down by team

## Using filters

You can filter the Overview report by:

* Date range
* Team
* Repository

Use the filter bar at the top of the page, or press `F` to add a filter.

## Time granularity

The default granularity is **week**. Change it by pressing `G` or clicking the granularity pill.
