- MCP servers are third-party tools (from anyone)
- No central governance of MCP permissions
- No audit of what MCPs access
- No control over which teams use which MCPs
- Security risk: Malicious MCP could expose customer data

**Example Threat Scenario:**

**MindWeave Solution:** Zero-trust MCP governance
