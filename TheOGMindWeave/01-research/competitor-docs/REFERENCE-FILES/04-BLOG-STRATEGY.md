### Weave's Content Characteristics

**Tone**:
- Professional but personable

## Content Tools & Templates

### Article Template

**Title Formula**:
- How-to: "How to [action] [outcome]"
- Comparison: "[Product] vs [Competitor]"
- Framework: "[Number] [Noun] for [Goal]"
- Problem: "The problem with [Traditional approach]"

**Structure**:
1. Hook (pain point or surprising stat)
2. Context (why this matters now)
3. Core concept (what you're teaching)
4. Application (how to use it)
5. Results (what success looks like)
6. CTA (try MindWeave / read next)

### Research/Data Template

- Survey or analysis (original research builds authority)
- Case study metrics (quantified results)
- Benchmark data (industry comparisons)
- Tool guides (with MindWeave integration examples)
