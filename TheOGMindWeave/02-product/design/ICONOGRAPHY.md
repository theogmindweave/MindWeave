## Overview

The MindWeave iconography system provides a consistent visual language for representing actions, objects, and concepts throughout the platform. Built on the Lucide icon library, this system ensures accessibility, scalability, and visual harmony across all interfaces.

## 7. Icon Components

### 7.1 Icon Component API

```typescript
interface IconProps {
  name: string;              // Icon name from Lucide
  size?: 'xs' | 'sm' | 'md' | 'lg' | 'xl' | '2xl' | number;
  color?: string;            // Tailwind color class or hex
  strokeWidth?: number;      // Default: 2
  className?: string;        // Additional classes
  'aria-label'?: string;     // For standalone icons
  'aria-hidden'?: boolean;   // For decorative icons
}

// Usage
<Icon name="settings" size="md" color="slate-500" />
<Icon name="plus" size={20} aria-hidden />
<Icon name="trash-2" aria-label="Delete item" />
```

### 7.2 Icon Button Component

```typescript
interface IconButtonProps {
  icon: string;
  size?: 'sm' | 'md' | 'lg';
  variant?: 'default' | 'ghost' | 'destructive';
  disabled?: boolean;
  loading?: boolean;
  tooltip?: string;
  onClick?: () => void;
  'aria-label': string;     // Required for accessibility
}

// Usage
<IconButton
  icon="edit"
  size="md"
  tooltip="Edit team"
  aria-label="Edit team"
  onClick={handleEdit}
/>

<IconButton
  icon="trash-2"
  variant="destructive"
  tooltip="Delete"
  aria-label="Delete team"
  onClick={handleDelete}
/>
```
