### 1.2 CSS Implementation

```css
/* Mobile First: Base styles for smallest screens */
.component {
  /* xs (0px+) - default mobile styles */
  padding: 16px;
  font-size: 14px;
}

/* Tablet and larger */
@media (min-width: 640px) {
  .component {
    /* sm (640px+) */
    padding: 20px;
  }
}

@media (min-width: 768px) {
  .component {
    /* md (768px+) */
    padding: 24px;
  }
}

@media (min-width: 1024px) {
  .component {
    /* lg (1024px+) */
    padding: 32px;
  }
}

@media (min-width: 1280px) {
  .component {
    /* xl (1280px+) */
  }
}

@media (min-width: 1536px) {
  .component {
    /* 2xl (1536px+) */
  }
}
```
