```
┌─────────────────────────────────────────────────────────────────────────┐
│                    LIQUIDATION WATERFALL                                 │
│                   Example: $50M Exit (Post-Series A)                     │
│                                                                         │
│   STEP 1: Series A Preference (1x non-participating)                    │
│   └── Series A invested: $6M × 1x = $6M                                 │
│                                                                         │
│   STEP 2: Seed Preference (1x non-participating)                        │
│   └── Seed invested: $2M × 1x = $2M                                     │
│                                                                         │
│   STEP 3: Pre-Seed Preference (1x non-participating)                    │
│   └── Pre-Seed invested: $0.5M × 1x = $0.5M                             │
│                                                                         │
│   REMAINING FOR PRO-RATA: $50M - $8.5M = $41.5M                         │
│                                                                         │
│   CONVERSION ANALYSIS:                                                  │
│   ─────────────────────────────────────────────────────────────────     │
│   Series A: Take pref ($6M) or convert to 16.13% ($8.07M) → CONVERT     │
│   Seed: Take pref ($2M) or convert to 16.13% ($8.07M) → CONVERT         │
│   Pre-Seed: Take pref ($0.5M) or convert to 9.68% ($4.84M) → CONVERT    │
│                                                                         │
│   ALL CONVERT TO COMMON → Pro-rata distribution of $50M                 │
│                                                                         │
│   FINAL DISTRIBUTION:                                                   │
│   ├── Founders: 51.61% × $50M = $25,805,000                             │
│   ├── Pre-Seed: 9.68% × $50M = $4,840,000                               │
│   ├── Seed: 16.13% × $50M = $8,065,000                                  │
│   ├── Series A: 16.13% × $50M = $8,065,000                              │
│   └── Options: 6.45% × $50M = $3,225,000                                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 7. Anti-Dilution Provisions

### 7.1 Protection Mechanism

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    ANTI-DILUTION PROVISIONS                              │
│                                                                         │
│   STANDARD: Broad-Based Weighted Average                                │
│                                                                         │
│   Formula:                                                              │
│   New Conversion Price = Old Price × (A + B) / (A + C)                  │
│                                                                         │
│   Where:                                                                │
│   A = Outstanding shares before new issue                               │
│   B = Shares that would have been issued at old price                   │
│   C = Shares actually issued at new (lower) price                       │
│                                                                         │
│   EXAMPLE: Down Round Scenario                                          │
│   ─────────────────────────────────────────────────────────────────     │
│   Seed price: $1.00/share                                               │
│   Down round price: $0.50/share                                         │
│   Outstanding shares: 10,000,000                                        │
│   New shares issued: 4,000,000                                          │
│                                                                         │
│   B = $2,000,000 / $1.00 = 2,000,000 shares                             │
│   C = 4,000,000 shares                                                  │
│                                                                         │
│   New Conversion = $1.00 × (10M + 2M) / (10M + 4M)                       │
│                  = $1.00 × 12M / 14M                                     │
│                  = $0.857/share                                          │
│                                                                         │
│   Seed investors get partial protection (not full ratchet to $0.50)     │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```
