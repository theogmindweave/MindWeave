## 8. Term Sheet Expectations

### 8.1 Key Terms

| Term | Expectation | Flexibility |
|------|-------------|-------------|
| Valuation | $6-8M pre | Moderate |
| Option Pool | 10-15% | Standard |
| Liquidation Pref | 1x non-participating | Standard |
| Board Seats | 3 (2 founders + 1 investor) | Low |
| Pro-rata Rights | Yes for lead | Standard |
| Information Rights | Quarterly updates | Standard |

### 8.2 Terms to Avoid

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    TERM SHEET RED FLAGS                                  │
│                                                                         │
│   AVOID                           WHY                                   │
│   ─────────────────────────────────────────────────────────────────     │
│   >1x liquidation preference      Dilutes founder upside                │
│   Participating preferred         Double-dip on returns                 │
│   Full ratchet anti-dilution      Punitive in down rounds               │
│   >2 board seats for investors    Loss of control too early             │
│   Excessive founder vesting       Should have credit for time           │
│   Broad protective provisions     Limits operational flexibility        │
│   Redemption rights               Creates debt-like obligation          │
│   Dividends (cumulative)          Rarely appropriate for seed           │
│                                                                         │
│   ACCEPTABLE                                                            │
│   ─────────────────────────────────────────────────────────────────     │
│   1x non-participating preferred  Standard                              │
│   Broad-based weighted avg        Fair anti-dilution                    │
│   Standard protective provisions  Normal governance                     │
│   Pro-rata rights                 Investor alignment                    │
│   Information rights              Transparency                          │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```
