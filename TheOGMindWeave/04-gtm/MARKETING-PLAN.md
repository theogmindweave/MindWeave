## 8. Brand Guidelines

### 8.1 Brand Voice

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    BRAND VOICE GUIDELINES                                │
│                                                                         │
│   VOICE ATTRIBUTES                                                      │
│   ─────────────────────────────────────────────────────────────────     │
│                                                                         │
│   EXPERT but not arrogant                                               │
│   • "Based on our analysis of 1,000 Claude deployments..."              │
│   • NOT: "We know everything about AI governance..."                    │
│                                                                         │
│   CLEAR but not simplistic                                              │
│   • "MindWeave shows you exactly what your team is building"            │
│   • NOT: "Easy AI governance for dummies"                               │
│                                                                         │
│   CONFIDENT but not aggressive                                          │
│   • "The governance platform built for Claude"                          │
│   • NOT: "The only choice for serious companies"                        │
│                                                                         │
│   HELPFUL but not pushy                                                 │
│   • "Ready to see your AI usage? Start a free trial"                    │
│   • NOT: "You're losing money every day without us!"                    │
│                                                                         │
│   TONE BY CONTEXT                                                       │
│   ─────────────────────────────────────────────────────────────────     │
│   • Website: Professional, confident                                    │
│   • Blog: Educational, friendly                                         │
│   • Social: Conversational, engaging                                    │
│   • Sales: Consultative, solution-focused                               │
│   • Support: Helpful, patient                                           │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 8.2 Messaging Don'ts

| Avoid | Instead Say |
|-------|-------------|
| "Revolutionary AI platform" | "Purpose-built for Claude governance" |
| "Synergy" / "Leverage" | "Work better together" / "Use" |
| "Best-in-class" | Specific capability claims |
| Fear-based messaging | Value-based messaging |
| Excessive superlatives | Specific, provable claims |
