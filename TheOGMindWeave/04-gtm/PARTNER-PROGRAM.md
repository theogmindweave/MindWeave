## 7. Partner Operations

### 7.1 Deal Registration

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    DEAL REGISTRATION PROCESS                             │
│                                                                         │
│   STEP 1: REGISTRATION                                                  │
│   └── Partner submits via portal:                                       │
│       • Customer name and contact                                       │
│       • Opportunity description                                         │
│       • Estimated deal size                                             │
│       • Expected close date                                             │
│                                                                         │
│   STEP 2: VALIDATION (24-48 hours)                                      │
│   └── MindWeave reviews:                                                │
│       • No existing opportunity                                         │
│       • Customer fit verification                                       │
│       • Partner capacity confirmation                                   │
│                                                                         │
│   STEP 3: APPROVAL                                                      │
│   └── Partner receives:                                                 │
│       • Registration confirmation                                       │
│       • Protection period (30-90 days)                                  │
│       • Discount level confirmation                                     │
│                                                                         │
│   STEP 4: DEAL SUPPORT                                                  │
│   └── MindWeave provides:                                               │
│       • Pre-sales support                                               │
│       • Demo assistance                                                 │
│       • Proposal templates                                              │
│                                                                         │
│   STEP 5: CLOSE & PAYOUT                                                │
│   └── Upon deal close:                                                  │
│       • Commission calculated                                           │
│       • Payout within 30 days of payment                                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```
