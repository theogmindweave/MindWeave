## SECTION 1: FOUNDER'S DAILY RHYTHM (Non-Negotiable)

### 1.1 Daily Schedule (Protect These Hours)

**6:00 AM - 7:00 AM: MORNING RITUAL (Non-Negotiable)**
```
├─ Exercise (30 min): Running, gym, yoga
│  └─ WHY: Clear mind, energy for decision-making
│
├─ Review (20 min): Check overnight emails/Slack
│  └─ Crisis scan: Any fires that need immediate attention?
│
├─ Planning (10 min): Today's 3 biggest wins
│  └─ What must happen today for progress?
│
└─ Breakfast: Protein + carbs, coffee, water
```

**7:00 AM - 9:00 AM: DEEP WORK (Do NOT Schedule Meetings)**
```
├─ Priority: Strategic thinking only
├─ Tasks: Partnership strategy, positioning, financial modeling
├─ Output: Decisions that only founder can make
├─ Location: Private (home or quiet space, NOT office)
└─ Rule: 2 hours completely uninterrupted
```

**9:00 AM - 12:00 PM: TEAM EXECUTION**
```
├─ 9:00 AM - 9:15 AM: All-hands standup (15 min)
│  ├─ Wins yesterday, 3 focuses today
│  ├─ Blocker removal (founder solves obstacles)
│  └─ Owner: Founder runs this DAILY
│
├─ 9:30 AM - 10:30 AM: Strategic initiative (60 min)
│  ├─ Rotate weekly focus (partnerships, product, sales, finance)
│  └─ Owner: Founder + 1-2 key leaders
│
└─ 10:30 AM - 11:30 AM: Customer/Partnership meetings
   ├─ Customer calls (5/week minimum)
   ├─ Partnership calls (Anthropic, partners)
   └─ Owner: Founder (personally, every week)
```

**12:00 PM - 1:00 PM: LUNCH (PROTECTED)**
- Don't skip. Refuel for afternoon energy.

**1:00 PM - 2:00 PM: OPERATIONAL EXECUTION**
```
├─ Hire/onboarding (when recruiting)
├─ Board prep/investor meetings (when fundraising)
├─ Financial review (weekly deep-dive every Friday)
└─ Admin/tactical work (only if urgent)
```

**2:00 PM - 5:00 PM: DELEGATION ZONE**
```
├─ Own team leads execution on:
│  ├─ Product development
│  ├─ Sales pipeline work
│  ├─ Customer success
│  ├─ Operations
│  └─ Marketing
│
└─ Founder's role: Remove blockers, make decisions, guide direction
```

**5:00 PM - 6:00 PM: FOUNDER REFLECTION & NEXT DAY PREP**
```
├─ Review: Did today's 3 wins happen?
├─ Wins: What went right? Why?
├─ Blocks: What stopped us? How to fix?
├─ Tomorrow: Set 3 new wins for next day
└─ Decision log: Log major decisions made + rationale
```

## SECTION 2: WEEKLY RHYTHM

**MONDAY - Focus Setting**
- All-hands (10 AM): Share week's strategic priority
- Deep work: Plan week's 3 biggest wins
- Customer calls: Start customer engagement (2-3 calls)

**TUESDAY - WEDNESDAY - Execution**
- Deep work: 7-9 AM (strategic thinking - rotating focus by week)
- Sales/customer engagement (continuous)
- Team meetings and blockers (as needed)

**THURSDAY - Course Correction**
- Review: Are we tracking to weekly goals?
- Decision: Any pivots needed?
- Customers: High-value prospect calls

**FRIDAY - Metrics & Planning**
- 3-Hour Financial Review (1-4 PM)
  - Revenue pipeline review
  - Burn rate check
  - Unit economics trending
  - Forecast for next 12 weeks
- Decision log review (what went right/wrong this week?)
- Next week planning (confirm 3 big wins)

## SECTION 4: THE FOUNDER'S OPERATING SYSTEM

### What This Playbook Covers

This consolidated document includes:

- ✓ Mental models for founder thinking
- ✓ Sales process & scripts
- ✓ Financial planning & controls
- ✓ Customer success playbooks
- ✓ Team building & hiring roadmaps

- ✓ 100-day execution checklist (separate doc)

### What NOT to Over-Engineer

- Don't add meetings beyond what's scheduled
- Don't build process where execution matters more
- Don't hire before you have paying customers
- Don't optimize before you know what customers want
- Don't plan 2 years out (quarterly is enough)

### Founder Decision Framework

**Use the Council of Experts for major decisions:**

1. **Gather input from:**
   - Strategic advisor (GTM, business)
   - Product leader (customer, features)
   - Engineering lead (technical feasibility)
   - Growth/sales lead (revenue impact)
   - Researcher/data person (market signals)

2. **Log the decision:**
   - What are we deciding?
   - Why now?
   - Options considered
   - Which option selected
   - Rationale
   - Expected impact
   - Sign-offs from council

3. **Review monthly:**
   - Was the decision right?
   - What would we do differently?
   - Update assumptions for next month

## SECTION 6: SUCCESS METRICS (Review Weekly)

**Revenue Metrics:**
- [ ] Weekly MRR (moving target)
- [ ] Customer acquisition rate (deals closed)
- [ ] CAC (cost per customer acquired)
- [ ] LTV (lifetime value per customer)
- [ ] LTV:CAC ratio (target 4:1 or better)

**Team Metrics:**
- [ ] Team size (hiring targets by month)
- [ ] Morale/engagement (quarterly survey)
- [ ] Turnover rate (should be <5% annually)

**Product Metrics:**
- [ ] Feature completion (roadmap progress)
- [ ] Bug severity (critical/high to fix this week)
- [ ] Customer feedback (NPS, feature requests)

**Financial Metrics:**
- [ ] Monthly burn rate (expense trend)
- [ ] Runway (months until cash depleted)
- [ ] Gross margin (if applicable)
- [ ] Forecast accuracy (actual vs. budget)
