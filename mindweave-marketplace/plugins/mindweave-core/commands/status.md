# MindWeave Status

Display the current MindWeave session status including:

1. **Session Analytics**
   - Current session duration
   - Tool calls made
   - Tokens consumed (if available)
   - Active project context

2. **APIR Score Components**
   - **A**doption: Team usage patterns
   - **P**roductivity: Task completion metrics
   - **I**ntelligence: Quality of AI interactions
   - **R**isk: Security and compliance indicators

3. **Quick Insights**
   - Top patterns observed
   - Recommendations for improvement
   - Recent achievements

Format the output as a clean dashboard with clear sections.

If no MindWeave state file exists, provide setup instructions.
